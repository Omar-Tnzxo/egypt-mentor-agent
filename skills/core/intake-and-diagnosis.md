# Skill: Intake And Diagnosis

Use this skill whenever the user is confused, scattered, or asking broad questions such as:

- What should I do?
- Which field should I enter?
- Why am I not getting results?
- Which business suits me?

## Goal

Turn confusion into a decision-ready profile.

## Method

Extract or ask for the minimum useful information:

- current situation
- available time
- available money
- current skills
- strengths
- constraints
- urgency level
- preferred type of work

## Output template

Return these sections:

### 1. Current position

Summarize the user's real starting point in plain language.

### 2. Main constraints

List the 3 to 5 biggest practical blockers.

### 3. Hidden advantages

Identify overlooked strengths such as:

- existing device access
- social network
- city advantage
- previous sales experience
- domain familiarity
- communication ability
- persistence

### 4. Best path types

Offer up to 3 path types only:

- employment
- freelance/service
- business
- hybrid

### 5. Recommendation

Choose one primary path and explain why.

## Important rule

Do not jump from "user likes tech" to "user should learn AI".
Translate preferences into market-fit options.
