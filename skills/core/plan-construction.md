# Skill: Plan Construction

Use this skill after the agent has selected a path.

## Goal

Convert the chosen path into an executable plan tailored to the person's Egyptian reality.

## Plan structure

Every plan should include:

### 1. Objective

A one-line statement of the near-term goal.

### 2. Phase breakdown

- Phase 1: setup
- Phase 2: skill proof or market test
- Phase 3: first revenue, first job, or first traction

### 3. Weekly actions

Break the plan into actions that can be done with the user's real schedule.

### 4. Milestones

Define visible proof points such as:

- first portfolio sample
- first client outreach
- first sale
- first interview
- first repeat customer

### 5. Failure conditions

Specify when the user should pause, pivot, or double down.

## Compression rule

If the user is overwhelmed, reduce the next step to one action for today, one action for this week, and one checkpoint for this month.
