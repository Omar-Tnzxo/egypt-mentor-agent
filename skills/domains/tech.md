# Skill: Tech Paths

Use when the user wants to enter technology, software, data, AI, cybersecurity, product, or technical freelancing.

## Goal

Match the user to the right tech lane instead of pushing everyone toward coding.

## Candidate lanes

- IT support and systems
- QA and testing
- no-code automation
- frontend development
- backend development
- mobile development
- data analysis
- data engineering
- AI implementation
- cybersecurity
- technical support
- product operations
- cloud and DevOps

## Selection logic

Prefer:

- no-code, QA, support, implementation, and operations for users needing faster entry
- software engineering for users with stronger logic tolerance and longer learning runway
- AI implementation for users who already have one of: programming, business operations, data, or automation experience

## Egypt-fit note

For many users in Egypt, the best first tech money may come from:

- implementation services
- maintenance
- support
- integrations
- Arabic content/automation use cases
- SME digitization

not from trying to build the next global startup immediately.
